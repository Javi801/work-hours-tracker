# Architecture

## Current architecture approach
This repository starts with a lightweight, folder-first architecture designed for gradual implementation.

## Top-level layout
- `app/mobile/android/` - Android platform placeholder
- `app/mobile/ios/` - iOS platform placeholder
- `app/mobile/src/` - Shared source structure placeholders
- `app/mobile/assets/` - Static asset placeholders
- `app/mobile/tests/` - Test folder placeholder
- `docs/` - Project planning and design docs

## Notes
- No runtime code is implemented yet
- No external services are configured
- Structure is intentionally minimal to keep future choices flexible
